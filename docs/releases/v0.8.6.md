# db-mcp v0.8.6

Release date: 2026-03-27

## Summary

`v0.8.6` is a patch release on top of `v0.8.5`. It adds preset-guided API setup in the
connection wizard, moves bundled connector presets into a dedicated templates directory, and adds
end-to-end coverage for the new API onboarding flows.

## Highlights

- Added preset-guided API setup so known providers like Jira and Metabase can drive auth defaults,
  required env vars, and generated `connector.yaml`.
- Moved bundled connector presets into `packages/core/src/db_mcp/templates/connectors`.
- Added Playwright coverage for API preset hydration, saved env rows, floating alerts, and the
  workspace `+ New` path.

## Breaking changes

- None

## Features

- Added first-class API preset descriptors and template-backed connector generation in the wizard.
- Added connection-local env var persistence from the UI, including masked saved-state handling.
- Added Playwright coverage for API preset flows and existing connection hydration.

## Fixes

- Fixed the UI/`connector.yaml` disconnect for preset-backed API connections by using preset
  metadata as the source of truth.
- Fixed existing Jira and Metabase-style connections so reopening the wizard restores the correct
  preset and saved env rows.
- Fixed the workspace `+ New` control so it consistently opens a fresh wizard route.
- Fixed saved env rows so persisted secrets no longer show an `Add` action.

## Security

- None

## Upgrade notes

- Existing API connections backed by bundled presets should reopen with their preset selected and
  saved env vars shown as already stored.
- Connector preset assets now live under `packages/core/src/db_mcp/templates/connectors`.
