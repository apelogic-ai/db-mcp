# db-mcp v0.8.7

Release date: 2026-03-27

## Summary

`v0.8.7` is a patch release on top of `v0.8.6`. It fixes the packaged binary so bundled API
presets are included at runtime again.

## Highlights

- Restored bundled API presets in packaged builds of the connection wizard.
- Fixed the packaging path so Jira, Metabase, Dune, and Superset presets are available at runtime.

## Breaking changes

- None

## Features

- Added a packaging regression test covering connector template inclusion in the PyInstaller spec.

## Fixes

- Fixed `packages/core/db-mcp.spec` so `db_mcp/templates` is bundled into the PyInstaller binary.
- Fixed released app builds where the API preset dropdown only showed `Custom` because the
  connector template YAML files were not present inside the packaged binary.

## Security

- None

## Upgrade notes

- Upgrade packaged installations to `v0.8.7` to restore the built-in API preset list.
- Repo-based development runs were not affected because the templates exist in the source tree.

## Validation

- `cd packages/core && uv run ruff check . --fix`
- `cd packages/core && uv run pytest tests/ -v`
- `cd packages/core && uv run pyinstaller --clean --noconfirm db-mcp.spec`
- `cd packages/core && uv run pyi-archive_viewer -r -b dist/db-mcp | rg 'db_mcp/templates/connectors'`

Result:

- Core suite: `1267 passed, 1 warning`
- Packaged binary inspection: bundled connector templates present
